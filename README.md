# Sail-Boat-AI

Github Link: [Sail-Boat-AI](https://github.com/AR-Rakha/Sail-Boat-AI)

## Opening The Program

1. Open the Final folder  
2. Run Main.exe

## Controls

r: reset

### Player boat (Green boat)

a: turn left  
d: turn right  
  
w: stop sailing

### AI boat (Overriding AI actions) (Pink boat)

arrow left: turn left  
arrow right: turn right  
  
arrow up: stop sailing

## Task

Reach the points on the screen in order  
  
***Notice that point number 1 will always be in the center where the sailboats start***
